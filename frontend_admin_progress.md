# 管理端（Admin）前端开发进度

> 更新时间：2026-05-26

## 已完成

### 1. 项目初始化
- `frontend/admin/` 独立项目，Vite + Vue 3 + TypeScript

- vue-router 5.x（与 H5 端对齐）

- 开发端口 3001，API 代理 `/api` → `localhost:8080`

### 2. Mock 数据层（MSW）— 已移除

## 已知问题 & 修复

### MSW 首次请求穿透问题（已修复）
- **现象**: 页面加载时弹出"网络错误"，Vite 终端显示 proxy error
- **原因**: MSW Service Worker 激活需要时间，首个 profile 请求穿透到 Vite 代理
- **修复**: axios 拦截器支持 `X-Silent` 标记，路由守卫的 profile 检查静默处理
- **涉及文件**: `api/index.ts`、`api/auth.ts`、`stores/auth.ts`、`router/index.ts`
