# 商品简介（描述）H5 端展示 — 实施计划

> **状态**：已批准，待实施
> **制定日期**：2026-06-30
> **登记位置**：`progress.md` →「已知问题 & 待开发」

## 背景

Admin 端新增/编辑商品时已有「描述」输入框（`frontend/admin/src/views/ProductPage.vue:325`），后端 `product` 表、`Product` 实体、Mapper、详情接口都已带 `description` 字段。但 H5 端拿到描述却未渲染——`SkuSelectorPopup.vue:17` 预留了空的 `.sku-desc` 占位，首页商品卡片也没展示。本计划让顾客能在 H5 看到商品简介。

链路已逐段确认，改动点少（后端 SQL 无需动，仅返回 Map 加字段；前端两处渲染 + 一处类型补字段）。

### 现状链路核查

| 环节 | 现状 | 是否需改 |
|---|---|---|
| Admin 端「描述」输入 | `ProductPage.vue:325` textarea 已存在 | 否 |
| 后端 `product` 表 / `Product` 实体 | 已有 `description` 字段 | 否 |
| 后端 Mapper `selectPageList` | `SELECT p.*`，`productResultMap`（`ProductMapper.xml:9`）已映射 description | 否 |
| 后端详情接口 `GET /api/app/products/{id}` | `AppProductController.getDetail()` 已返回 description（第 75 行） | 否 |
| 后端列表接口 `GET /api/app/products` | `toAppMap()` 返回的 Map **未含** description | **改** |
| H5 `Product` 类型 | 未定义 `description` 字段 | **改** |
| H5 首页商品卡片 | 未渲染描述 | **改** |
| H5 SKU 选择弹窗 | `SkuSelectorPopup.vue:17` 预留空 `.sku-desc` 占位，未填充 | **改** |

## 实施步骤（执行时按序）

### 1. 后端：列表接口回传 description

**文件**：`backend/src/main/java/com/petshop/order/controller/app/AppProductController.java`

- `toAppMap()`（约 148 行）返回的 Map 中新增 `description` 字段：
  ```java
  "description", p.getDescription() != null ? p.getDescription() : "",
  ```
- **无需改 SQL / Mapper**：`selectPageList` 已 `SELECT p.*`，`productResultMap` 已映射 description，`Product` 实体已有该字段。
- 详情接口 `getDetail()`（72 行）已返回 description，不动。

> 与详情接口风格一致（详情也用 `description != null ? description : ""` 兜底空串）。

### 2. 前端 H5：类型补字段

**文件**：`frontend/h5/src/types/index.ts`

- `Product` 接口（17-26 行）新增 `description: string`（后端已兜底为空串）。

### 3. 前端 H5：首页商品卡片展示简介

**文件**：`frontend/h5/src/components/home/ProductCard.vue`

- 在 `.product-name`（10 行）下方加一行描述，单行省略，灰色小字：
  ```vue
  <div v-if="product.description" class="product-desc">{{ product.description }}</div>
  ```
- 配套样式：`font-size: 12px; color: $text-muted;` 单行 `text-overflow: ellipsis; white-space: nowrap; overflow: hidden;`，留小间距。
- 仅在有描述时显示，避免空行撑高无描述的卡片。

### 4. 前端 H5：SKU 选择弹窗填充预留占位

**文件**：`frontend/h5/src/components/product/SkuSelectorPopup.vue`

- 第 17 行空 `<div class="sku-desc"></div>` 改为：
  ```vue
  <div v-if="product?.description" class="sku-desc">{{ product.description }}</div>
  ```
- `.sku-desc` 样式已存在（175 行：12px 灰字，min-height 18px），保留即可。描述较长时改为 2 行省略（`-webkit-line-clamp: 2`），避免弹窗头部过高。
- `product` 是 `ProductDetail` 类型，已含 description（HomePage `openSpecPicker` 调详情接口拿到）。
